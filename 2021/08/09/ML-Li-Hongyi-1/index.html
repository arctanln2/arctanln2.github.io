机器学习包括以下三类

  • Regression：输出是一个值
  • Classification：输出是一个类别，一个选择，离散的
  • Structured Learning：机器自己创造东西

机器学习三步走——训练

  • Step 1：function with unknown
  • Step 2：define loss from training data
  • Step 3：optimization

Loss

MAE

MSE

Optimization

Gradient Descent

hyperparameters：$\eta$ 学习率

问题：最后结果是局部最优，不一定能找到全局最优

Does local minima truly cause the problem？过拟合？

Function 模型

模型的修改来源于domain knowledge

  • Linear Model

Model Bias：来自于model的限制

Piecewise Linear curve：分段线性函数

Sigmoid函数去逼近Hard Sigmoid

$y=b+\sum_{i}c_{i}sigmoid(b_{i}+\sum_{j}w_{ij}x_{j})$

利用向量可以简写成

$$y=b+c^{T}\sigma(b+Wx)$$

Unknown parameters：$W,x,c,b$，可以写成一列$\theta$

$$\theta^{t+1} \leftarrow \theta^{t}-\eta g$$

Update：每个batch弄一下

Epoch：所有batch弄完

Rectified Linear Unit (ReLu): c $max(0,b+wx_{1})$

Activation Function: Sigmoid, ReLu. Linear

Layer number: another hyperparameter

Neuron：神经元
