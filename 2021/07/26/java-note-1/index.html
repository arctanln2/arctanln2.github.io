Java介绍

java编写分布式、复杂的系统，C++的体系

能干什么

手机、网站、服务器、桌面

历史

1972年C诞生：指针，内存管理

1982年C++诞生：面向对象，图形领域，游戏

1995年java诞生：可移植性

基于Java开发了巨多平台，系统，工具，包括构建工具、应用服务器、Web开发、开发工具等。

特性

  • 简单性
  • 面向对象
  • 可移植性
  • 高性能
  • 分布式
  • 动态性
  • 多线程
  • 安全性
  • 健壮性

三大版本

  • JavaSE：标准板（桌面程序，控制台开发。。）
  • JavaME：嵌入式开发
  • JavaEE：企业级开发（web端，服务器开发。。）

JDK、JRE、JVM

JDK：Java Developmeent Kit

JRE：Java Runtime Environment

JVM：JAVA Virtual Machine
